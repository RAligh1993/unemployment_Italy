# 🇮🇹 Italian Unemployment Nowcasting

Real-time unemployment forecasting for Italy using machine learning.

## 🚀 Features

- **Models**: NAIVE, MA3, Ridge Regression, MIDAS
- **Data**: Monthly unemployment, Financial indicators, Google Trends
- **Period**: 2016-2024
